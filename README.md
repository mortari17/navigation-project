# 📱 Projeto: Navegação com Navigation Compose

## 📌 Descrição do Projeto

Este projeto tem como objetivo recriar a evolução de uma aplicação Android utilizando **Jetpack Compose**, com foco na implementação de navegação entre telas usando **Navigation Compose**.

A construção foi feita de forma incremental, seguindo o histórico de commits proposto, partindo de uma `MainActivity` limpa até a implementação completa da navegação entre telas.

O objetivo principal foi compreender na prática como funciona a navegação em aplicações modernas Android utilizando Compose.

---

## 🚀 Funcionalidades

- Navegação entre múltiplas telas
- Gerenciamento de rotas com `NavController`
- Interface declarativa com Compose
